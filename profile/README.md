# Anthroid Dev Team
江苏大学计算机科学与技术专业科创团队，深耕**人格化AI**领域研发，为江苏大学大学生计算机设计大赛「Anthroid人格化AI」作品专属开发团队。

### 团队释义
团队名称由**anthro（类人/拟人）+ oid（类/形）** 组合而来，与核心作品Anthroid人格化AI的创作理念高度契合，以技术探索AI的人格化表达与落地可能。

### 团队构成
共5名计算机科学与技术专业学子，采用**4名大一+1名大二**的阶梯式配置，兼顾新锐创新活力与专业实践经验，分工协作完成算法实现、功能开发、项目统筹等全流程研发工作。

### 团队理念
以赛促学、以练践知，立足课堂专业知识，深耕AI技术实践，致力于打造有温度、可落地的人格化AI作品。
